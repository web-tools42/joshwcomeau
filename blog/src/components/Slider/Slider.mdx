# Slider

> A simple slider component. Can be used to select within a range of numbers.

<PropsTable of={Slider} />

## Vertical

<Playground>
  <Value initial={0}>
    {({ value, setValue }) => (
      <div style={{ display: 'flex', height: 200}}>
        <Slider
          orientation="vertical"
          value={value}
          step={1}
          onChange={setValue}
        />
      </div>
    )}
  </Value>
</Playground>
